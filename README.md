## Getting Started

### Prerequisites

- Node.js 18 or newer
- npm or yarn
- Expo CLI

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/kids-tv-launcher.git
```

2. Install dependencies:
```bash
cd kids-tv-launcher
npm install
```

3. Start the development server:
```bash
npm run dev
```
